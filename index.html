DRIP aims to collect the models and provides libraries for fixed income valuation and adjustments, asset allocation and transaction cost analytics, and supporting libraries in numerical optimization and statistical learning.

DRIP is composed of four main libraries:

  • DRIP Fixed Income
  • DRIP Asset Allocation
  • DRIP Numerical Optimizer
  • DRIP Statistical Learning

Installation

Installation is as simple as building a jar and dropping into the classpath. There are no other dependencies.

DRIP Core Technical Specifications

  • Asset Allocation Library
  • Asset Backed Analytics
  • Fixed Income Analytics
  • Transaction Cost Analytics
  • XVA Analytics

DRIP Supporting Technical Specifications

  • Spline Builder Library
  • Numerical Optimization Library
  • Statistical Learning Library
  • Machine Learning Library
